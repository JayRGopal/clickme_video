CLICKTIONARY

A web-based application for exploring the atoms of object recognition.

Despite rapid advancements in Deep Convolutional Network (DCN) performance, they still lag behind humans in all but the most basic object recognition tasks. One explanation is that they do not base their decisions on the same features as humans. CLICKTIONARY gathers information about the most diagnostic features in objects according to humans, and uses this information in two ways.

  • First, to train a model for predicting the locations of these features in novel images.
  • Second, to modulate DCN activity with this information, which we find improves classification accuracy.

    • See below for a depiction of the application and real-time results on the impact of these maps on DCN recognition. For more details on the project motivating this application, please visit our [arxiv], currently under review for CVPR 2017.

    Real-time results

    Here we record VGG 16 accuracy on ImageNet Top-1 object classification after modulating its activity with a model that predicts Human realization maps. These realization maps are predicted based on where people click in images on the main page.
